This highly-scalable solution processes raw Uber trip logs, executing null-filtering and data type standardization. The pipeline logically tiers data using a medallion architecture, guaranteeing deterministic output routing into the PostgreSQL warehouse for downstream analytics.

Pipeline Workflow

01

Raw Data Ingestion

Raw CSV files securely staged in AWS S3 Data Lakes.

02

Cleaning & Standardization

Handling null values and correcting timestamps via Pandas/PySpark.

03

Silver Layer Processing

Validated schemas loaded into the processed Silver bucket.

04

Gold Layer Aggregation

Business dimensions aggregated for low-latency BI queries.

05

Serving & Analytics

Final metrics synced to PostgreSQL data warehouse powering dashboards.

Key Outcomes

  • Built an end-to-end pipeline processing 10,000+ daily records from S3 staging to BI presentation.
  • Implemented Medallion Architecture, enforcing schema governance and data lakehouse reliability.
  • Orchestrated automated scheduling via an 8-task Airflow DAG to execute and retry workflows.
  • Secured validity using dbt executing comprehensive SLA and referential integrity tests.
